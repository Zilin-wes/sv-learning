# 01 - SV 数据类型

## 1. logic 类型
- 四态数据类型：0, 1, X, Z   **为了模拟硬件**
- 可以替代传统的 reg 和 wire
- **推荐在模块内部统一使用 logic**
- data width数据位宽:user define
- **data width是user define的，默认为unsigned无符号数(可以声明变成signed)** time是例外-64位-unsigned
- **data width是16/32固定数值的，默认为signed有符号数(不能变成unsigned)**
- wire&var型（自动识别） assign/连接模块IO&always/initial/function/task

```systemverilog
logic [7:0] data;      // 8位logic变量,
logic flag;            // 单bit
logic [3:0] addr = 4'h0;  // 声明时初始化
logic signed [3:0] a;    //声明变成有符号数
```
## 2. bit 类型
- 二态数据类型：只有 0 和 1   **模拟软件**
- 没有 X/Z，仿真速度更快，占用内存更少
- 不能用于 DUT 的输入输出端口
- data width数据位宽:user define
- var型 always/initial/function/task
  
```systemverilog
bit [31:0] addr;       // 32bit，无X/Z
bit valid;             // 单bit
```

## 3. byte/shortint/int/longint 类型
- 二态:0和1
- data width:8/16/32/64
- signed
- var型 always/initial/function/task

## 4.数据类型转换
- 截断高位丢失
- 扩展要扩展最高位0/1: 2'b10-4'b1110
- 符号转换位和数都不变解读变化: 2'b11-unsigned:3-signed:-1
### 4.1静态转换（编译时确定）

- 类型转换 data_type'(var)
- 数据位宽转换 bit_size'(var)
- 符号转换 signed'(var)

### 4.2动态转换（执行时转换）
- $cast(des_var, source_var)
- 因为类的继承 父类指针指向不同子类而存在
```systemverilog
bit [7:0] v0;
byte v1;
int v2;

initial begin
    #50;
    v0 = 200;
    v1 = byte'(v0);    //类型转换
    v2 = int'(v1);
    $display("1: v0=%0h, v1=%0h, v2=%0h.", v0, v1, v2);
    v0 = 8'(v2);    //数据位宽转换
    $display("2: v0=%0h.", v0);
    v1 = signed'(2'b11);    //符号转换
    $display("3: v0=%0h, v1=%0h, v2=%0h.", v0, v1, v2);
    v2 = unsigned'(2'b11);
    $display("4: v0=%0h, v1=%0h, v2=%0h.", v0, v1, v2);

    $cast(v0,v1);    //动态转换
    $display("4: v0=%0h, v1=%0h, v2=%0h.", v0, v1, v2);
end
  ```
